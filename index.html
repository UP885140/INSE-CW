The Deadline Day web application is a time efficient application that helps both student and members of the general public by oraganising the books they have borrowed from the libary. This is done by providing them with a variety of useful services such as remotley renewing a loan book, providing a brief information about the books and giving reminders regarding book return deadlines.

Features Available

- View Loan history

- Renew Book

- Book Recommendation

- Update Personal Settings

- Set Deadline Reminders
